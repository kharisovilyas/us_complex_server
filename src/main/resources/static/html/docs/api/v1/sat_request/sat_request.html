API Документация

Каталог целей

GET-запрос по адресу, чтобы получить весь список каталога:

GET http://localhost:8081/api/v1/satrequest/catalog/get/all

Пример ответа:

[
    {
        "goalId": 352,
        "goalName": "Тольятти-3",
        "lat": 12.334,
        "lon": 98.123,
        "alt": -12.341
    },
    {
        "goalId": 353,
        "goalName": "Новосибирск",
        "lat": 2.334,
        "lon": 54.123,
        "alt": -52.341
    }
]

POST-запрос по адресу:

POST http://localhost:8081/api/v1/satrequest/catalog/update - позволяет добавить, обновить, удалить сущности в каталоге целей. Отправляется список.

Пример запроса:

[
    {
        "goalId": 452,
        "deleted": true,
        "goalName": "Северодвинск",
        "lat": 12.334,
        "lon": 98.123,
        "alt": -12.341
    },
    {
        "goalId": 453,
        "deleted": false,
        "goalName": "Москва",
        "lat": 12.334,
        "lon": 98.123,
        "alt": -12.341
    },
    {
        "goalName": "Новосибирск",
        "lat": 2.334,
        "lon": 54.123,
        "alt": -52.341
    }
]

Номера узлов

Заявки ДЗЗ

POST-запрос на добавление, удаление, обновление списком записей в таблице Заявок, осуществляется по запросу:

POST http://localhost:8081/api/v1/satrequest/request/update

Пример запроса:

[
    {
        "requestId": 553,
        "deleted": true,
        "catalog": {
            "goalId": 503,
            "goalName": "Новосибирск",
            "lat": 2.334,
            "lon": 54.123,
            "alt": -52.341
        },
        "orderId": 2,
        "priory": 3,
        "term": "2023-04-27T14:37:15",
        "time": "2022-02-24T14:37:15",
        "idNode": {
            "entryId": 12761,
            "idNode": 2
        },
        "filter": false
    },

    {
        "requestId": 554,
        "catalog": {
            "goalId": 453,
            "goalName": "Москва",
            "lat": 12.334,
            "lon": 98.123,
            "alt": -12.341
        },
        "orderId": 2,
        "priory": 2,
        "term": "2024-05-27T14:37:15",
        "time": "2024-04-24T14:37:15",
        "idNode": {
            "entryId": 12759,
            "idNode": 1
        },
        "filter": true
    },
    {
        "catalog": {
            "goalId": 506,
            "goalName": "Новосибирск",
            "lat": 2.334,
            "lon": 54.123,
            "alt": -52.341
        },
        "orderId": 2,
        "priory": 2,
        "term": "2024-05-27T14:37:15",
        "time": "2024-04-24T14:37:15",
        "idNode": {
            "entryId": 12802,
            "idNode": 3
        },
        "filter": true
    }
]

Удаляет запись с requestId = 553, обновляет запись с requestId = 554. И добавляет новую запись в таблицу.
