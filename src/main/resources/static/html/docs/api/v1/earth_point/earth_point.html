НП (Наземные пункты)

1) POST запрос к http://localhost:8081/api/v1/earth/update/byList позволяет добавить, обновить и удалить запись в таблицу НП.

1. Атрибуты (JSON запрос) вида:

[
        {
            “nameEarthPoint”: “sgdfs”,
            “longitude”: 90,
            “latitude”: 180
        },
        {
            “nameEarthPoint”: “Fc123”,
            “longitude”: 1,
            “latitude”: 2
        }
]

- добавляет два поля в таблицу EarthPoint.

2. Атрибуты (JSON запрос вида):

[
        {
            “nameEarthPoint”: “sgdfs”,
            “longitude”: 90,
            “latitude”: 180,
            “id”: 6552,
            “deleted”: true
        },
        {
            “nameEarthPoint”: “Fc123”,
            “longitude”: 1,
            “latitude”: 2,
            “id”: 6553,
            “deleted”: false
        }
]

- Позволяет удалить запись с id = 6552.

- И обновить запись в таблице с id = 6553 (При обновлении также можно не указывать поле “deleted”: false) Пример ниже:

[
        {
            ...
        },
        {
            “nameEarthPoint”: “Fc123”,
            “longitude”: 1,
            “latitude”: 2,
            “id”: 6553
        }
]
